Steps before re-installing Windows#

  1. Manually trigger last restic backup
  2. Sync Drives (Firefox account, Google Drive, Dropbox)
  3. Manually copy media folders to a backup location
    1. /Users/<user>Documents
    2. /Users/<user>Pictures
    3. /Users/<user>Desktop
    4. /Users/<user>Downloads
  4. Manually take screenshots of Dota Settings screens
  5. Backup saved games through GameSave-Manager
  6. Backup Steam games to a Library folder on another disk if it’s in C:
  7. Push git folders
  8. Prepare Windows installation USB using Installation Media Creation Tool
